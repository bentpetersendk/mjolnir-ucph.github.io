Databases

Databases are centralized in a shared folder accessible to all users, which helps reduce storage costs and eliminate duplicate copies. This database repository is not limited to Mjolnir users but is available to all Globe affiliated users across KU-Research-IT-hosted compute systems. It's important to note that this folder is managed by Bent Petersen and is set to read-only access for individual users.

The repository is located at: /datasets/globe_databases
